Toolbar

Posted on 2017-02-07

  1. 在布局中使用<android.support.v7.widget.Toolbar>引进一个Toolbar，并推荐使用<android.support.design.widget.AppBarLayout>包裹它

     <android.support.design.widget.AppBarLayout
    android:layout_width="match_parent"
    android:layout_height="wrap_content">
    <android.support.v7.widget.Toolbar
        xmlns:android="http://schemas.android.com/apk/res/android"
        android:id="@+id/toolbar"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:background="@color/colorPrimary">
    </android.support.v7.widget.Toolbar>
</android.support.design.widget.AppBarLayout>

  1. 在style.xml以及v21/style.xml中设置Application的theme: (继承AppCompatActivity,会默认显示Action bar,要隐藏掉)

    <!-- values/styles.xml: -->
<style name="AppTheme" parent="AppBaseTheme"></style>

<style name="AppBaseTheme" parent="Theme.AppCompat.Light.NoActionBar">
    <item name="colorPrimary">@color/colorPrimary</item>
    <item name="colorPrimaryDark">@color/colorPrimaryDark</item>
    <item name="colorAccent">@color/colorAccent</item>
    <!-- 设置Toolbar的标题颜色 -->
    <item name="android:textColorPrimary">@android:color/white</item>
</style>

<!-- values-v21/styles.xml: -->
<style name="AppTheme" parent="AppBaseTheme">
    <item name="android:statusBarColor">@color/colorPrimaryDark</item>
    <item name="android:windowTranslucentStatus">true</item>
</style>

    经过以上设置，在5.0以上和4.4都能符合Material Design。

  2. 在java代码中让Toolbar显示：

    private Toolbar mToolbar;
mToolbar = (Toolbar) findViewById(R.id.toolbar);
// 通过代码设置Toolbar [也能在xml中设置]
toolbar.setLogo(R.drawable.ic_launcher);
toolbar.setTitle("My Title");
toolbar.setSubtitle("Sub title");
setSupportActionBar(toolbar);
// Navigation Icon要设置在setSupoortActionBar之后才有作用
toolbar.setNavigationIcon(R.mipmap.ic_android);

  3. 菜单的创建：在res/menu/xx_menu.xml中定义

    <menu xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto">
    <!-- orderInCategory:排列顺序,越小越靠前 -->
    <!-- app:showAsAction:
                ifRoom 有空间就显示
                never 不显示,藏到三个点之中 -->
    <item 
        android:id="@+id/action_search"
        android:title="search"
        android:orderInCategory="90"
        android:icon="@mipmap/ic_search"
        app:showAsAction="ifRoom" />

  <item android:id="@+id/action_settings"
        android:title="settings"
        android:orderInCategory="100"
        android:icon="@mipmap/ic_settings"
        app:showAsAction="never"/>
</menu>

    在java代码中加入OnMenuItemClickListener监听者

    private Toolbar.OnMenuItemClickListener onMenuItemClick = new Toolbar.OnMenuItemClickListener() {
    public boolean onMenuItemClick(MenuItem menuItem) {
    switch (menuItem.getItemId()) {
      case R.id.action_search:
        break;
      case R.id.action_settings:
        break;
    }
    return true;
    }
};

    设置OnMenuItemClickListener的监听者：

    // Menu item click 的监听事件也要设置在setSupportActionBar之后才有作用
toolbar.setOnMenuItemClickListener(onMenuItemClick);

  4. Action Menu Item 的文字颜色设置问题

    按照上面的写法，Toolbar菜单中的文字颜色将会和Toolbar标题颜色一样

    • 首先在布局文件中给Toolbar加一条popupTheme属性：
      • app:popupTheme=”@style/ToolbarTheme.PopupOverlay”
    • 在style中创建该样式[v21-style中创建相同的代码]并如下定义:
    <style name="ToolbarTheme.PopupOverlay" parent="@style/ThemeOverlay.AppCompat.ActionBar">
    <item name="android:textColorPrimary">@color/black</item>
</style>
    • 若想更改Toolbar菜单的默认图标(三个点)，则在布局文件中给Toolbar加一条theme属性：
      • app:theme=”@style/ToolbarTheme”

    • 在style中创建该样式[v21-style中只需创建第一个]并如下定义:

      <style name="ToolbarTheme">
    <item name="actionOverflowButtonStyle">@style/ActionButton.Overflow.WeChat</item>
</style>

<style name="ActionButton.Overflow.WeChat" parent="android:style/Widget.Holo.Light.ActionButton.Overflow">
    <item name="android:src">@mipmap/ic_add</item>
</style>

    • 若想更改菜单出现位置，背景色，则在AppBaseTheme中加一个条目：

      <item name="actionOverflowMenuStyle">@style/MenuStyle</item>

    • 然后在style.xml中创建该样式：

      <style name="MenuStyle" parent="@style/Widget.AppCompat.Light.PopupMenu.Overflow">
    <item name="overlapAnchor">false</item>
    <item name="android:popupBackground">@color/colorPrimary</item>
    <item name="android:dropDownVerticalOffset">0dip</item>
</style>
      • 必须设置parent为Widget.AppCompat.Light.PopupMenu.Overflow，这是Toolbar菜单默认样式的父类。overlapAnchor设置菜单不遮盖Toolbar， popupBackground设置背景色， dropDownVerticalOffset设置菜单与标题栏的垂直间距
