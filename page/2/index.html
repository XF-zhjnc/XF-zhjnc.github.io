Java GC

Posted on 2017-04-14

STW

“stop-the-world”。STW会在任何一种GC算法中发生。STW意味着JVM因为要执行GC而停止了应用程序的执行。当Stop-the-world发生时，除了GC所需的线程以外，所有线程都处于等待状态，直到GC任务完成。GC优化很多时候就是指减少Stop-the-world发生的时间。
